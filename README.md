# PUBG 地图标记叠加层

PUBG（绝地求生）地图标记辅助工具，在游戏画面上叠加透明图层，显示地图关键地点标记（密室、撬棍房、熊洞等），支持多地图切换和快捷键控制。

## 功能

- **透明叠加层** — 覆盖在游戏窗口上方，鼠标点击穿透，不影响游戏操作
- **7 张地图** — 艾伦格、米拉玛、泰戈、维寒迪、帕拉莫、帝斯顿、荣都
- **多色标记** — 维寒迪区分红(密室)、紫(撬棍房)、绿(熊洞)，其他地图为红色标记
- **全局热键** — 在任何应用（包括 Electron/BitFun）焦点下均有效
- **右键菜单** — 系统托盘图标右键菜单，可切换显示、筛选地图池、退出
- **地图筛选** — 在「地图池」子菜单中勾选/取消地图，只有勾选的地图参与切换循环

## 使用方法

### 启动

双击 `run_crosshair.bat` 或直接运行 `Crosshair.exe`。

启动后系统托盘会出现一个红色圆点图标，表示程序正在运行。

### 快捷键

| 按键 | 功能 |
|:----:|:----:|
| `` ` ``（反引号）或 `F2` | 切换标记显示/隐藏 |
| `←`（左箭头） | 切换到上一张地图 |
| `→`（右箭头） | 切换到下一张地图 |

### 右键菜单

右键点击系统托盘图标弹出菜单：

- **地图池** — 子菜单，显示所有地图名，带复选框。勾选的地图才会在按 `←` `→` 时切换。取消勾选当前显示的地图会自动跳到下一个已勾选的地图。
- **显示标记 / 隐藏标记** — 切换叠加层显示状态
- **退出** — 退出程序

### 注意事项

- 游戏需设置为**无边框窗口模式**（Borderless Window）或窗口模式
- 叠加层默认置顶显示，不会影响游戏操作
- 首次运行可能需要安装 .NET Framework 4.0 或更高版本

## 编译方法

如果修改源码后需要重新编译，运行以下命令（需要 Visual Studio Command Prompt 或 .NET SDK）：

```batch
"%SystemRoot%\Microsoft.NET\Framework\v4.0.30319\csc.exe" /target:winexe /reference:System.dll /reference:System.Windows.Forms.dll /reference:System.Drawing.dll Crosshair.cs
```

## 坐标文件

`坐标/` 文件夹中包含 7 个 txt 文件，列出了所有标记点的坐标。格式：

```
序号. (X坐标, Y坐标) [颜色-类型]
```

坐标是原始图片像素坐标，程序会自动根据屏幕分辨率缩放。
